# Coding Fight Backend

A real-time 1v1 coding challenge platform backend built with Node.js, Express, TypeScript, and Socket.io.

##  Project Overview

Coding Fight is a gamified platform where users can:
- Compete in real-time 1v1 coding challenges
- Get AI-powered hints while solving problems
- Track their performance and statistics
- Compete on leaderboards

This is the **v1.0 Release** of the backend API.
